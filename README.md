# Custom Featured Products

  ## What I've done
  - Created a custom Shopify section for featured products
  - Added support for selecting products in the Theme Editor
  - Rendered product image, title, price, sale badge and "add to cart" button
  - Implemented add to cart with JavaScript without full page reload using Ajax API
  - Updated the cart icon after adding a product without page reload
  - Added optimistic UI update for "Add to cart" button
